mlos_bench.tests.services.remote.ssh.conftest.ssh_test_server(ssh_test_server_hostname: str, docker_compose_project_name: str, docker_services: Services) → Generator[SshTestServerInfo, None, None]¶

Fixture for getting the ssh test server services setup via docker-compose using pytest-docker.

Yields the (hostname, port, username, id_rsa_path) of the test server.

Once the session is over, the docker containers are torn down, and the temporary file holding the dynamically generated private key of the test server is deleted.
