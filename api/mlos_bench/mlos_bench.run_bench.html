mlos_bench.run_bench module¶

OS Autotune benchmark launcher without involving the optimizer. Used mostly for development/testing purposes.

See –help output for details.

usage: run_bench.py [-h] [--log LOG_FILE] [--log-level LOG_LEVEL]
                    --environment ENVIRONMENT
                    [--config-path CONFIG_PATH [CONFIG_PATH ...]]
                    [--globals GLOBALS [GLOBALS ...]] [--no-teardown]
                    --tunables TUNABLES [TUNABLES ...]

mlos_bench run_bench

options:
  -h, --help            show this help message and exit
  --log LOG_FILE        Path to the log file. Use stdout if omitted.
  --log-level LOG_LEVEL
                        Logging level. Default is 20 (INFO). Set to 10 for
                        debug, 30 for warnings only.
  --environment ENVIRONMENT
                        Path to JSON file with the configuration of the
                        benchmarking environment.
  --config-path CONFIG_PATH [CONFIG_PATH ...]
                        One or more locations of JSON config files.
  --globals GLOBALS [GLOBALS ...]
                        Path to one or more JSON files that contain additional
                        [private] parameters of the benchmarking environment.
  --no-teardown         Disable teardown of the environment after the
                        benchmark.
  --tunables TUNABLES [TUNABLES ...]
                        Path to one or more JSON files that contain values of
                        the tunable parameters.
