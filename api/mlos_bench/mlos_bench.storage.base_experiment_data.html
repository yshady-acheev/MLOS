abstract property results: DataFrame¶

Retrieve all experimental results as a single DataFrame.

Returns:
resultspandas.DataFrame

A DataFrame with configurations and results from all trials of the experiment. Has columns [trial_id, config_id, ts_start, ts_end, status] followed by config parameters and trial results. The latter can be NULLs if the trial was not successful.
