mlos_core.tests package¶

Common functions for mlos_core Optimizer tests.

mlos_core.tests.get_all_concrete_subclasses(cls: Type[T], pkg_name: str | None = None) → List[Type[T]]¶

Gets a sorted list of all of the concrete subclasses of the given class. Useful for dynamically enumerating expected test cases.

Note: For abstract types, mypy will complain at the call site. Use “# type: ignore[type-abstract]” to suppress the warning. See Also: https://github.com/python/mypy/issues/4717

mlos_core.tests.get_all_submodules(pkg: TypeAlias) → List[str]¶

Imports all submodules for a package and returns their names. Useful for dynamically enumerating subclasses.
