mlos_core.tests.optimizers.one_hot_test module¶

Tests for one-hot encoding for certain optimizers.

mlos_core.tests.optimizers.one_hot_test.data_frame() → DataFrame¶

Toy data frame corresponding to the configuration_space hyperparameters. The columns are deliberately not in alphabetic order.

mlos_core.tests.optimizers.one_hot_test.one_hot() → ndarray[Any, dtype[_ScalarType_co]]¶

One-hot encoding of the data_frame above. The columns follow the order of the hyperparameters in configuration_space.

Toy problem to test one-hot decoding.

Round-trip test for one-hot-encoding and then decoding a data frame.

Round-trip test for one-hot-decoding and then encoding of a numpy array.
