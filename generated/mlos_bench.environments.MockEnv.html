A collection of tunable parameters for all environments.

run() → Tuple[Status, Dict[str, float] | None]¶

Produce mock benchmark data for one experiment.

Returns:
(status, output)(Status, dict)

A pair of (Status, output) values, where output is a dict with the results or None if the status is not COMPLETED. The keys of the output dict are the names of the metrics specified in the config; by default it’s just one metric named “score”. All output metrics have the same value.
